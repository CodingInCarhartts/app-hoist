## Getting Started

1. Install dependencies with Bun:
   ```bash
   bun install
   ```

2. Start the development server:
   ```bash
   bun run dev
   ```

3. Open [http://localhost:{{port}}](http://localhost:{{port}}) in your browser.

## Building

To build for production:
```bash
bun run build
```

Preview the production build:
```bash
bun run preview
```

## Type Checking

Run type checking:
```bash
bun run check
```

Watch mode:
```bash
bun run check:watch
```
